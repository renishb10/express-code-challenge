# Express Coding Challenge

## Running this application

You can run the application by typing:

- `npm install`
- `sequelize db:migrate`
- `sequelize db:seed:all`
- `npm start`

## Testing this application

You can run the application by typing:

- `npm test`
- `npm run coverage`
